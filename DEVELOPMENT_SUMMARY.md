## 📊 项目概览

**FreeCDN** 是一个基于 GoEdge v1.3.9 的开源 CDN 系统，目标是"**一条命令在自己服务器搭起完整 CDN，无需文档阅读，无计费墙**"。

## 🏗️ 系统架构（四仓库模型）

```
┌─────────────────────────────────────────────────────────┐
│                    FreeCDN 系统架构                       │
├─────────────────────────────────────────────────────────┤
│
│ ┌──────────────────┐                ┌──────────────────┐
│ │  freecdn-admin   │                │ freecdn-api      │
│ │ (管理后台 Web)    │────gRPC 8003──→│ (gRPC API 服务)  │
│ │ 端口: 7788        │  endpoint      │ 端口: 8003       │
│ └──────────────────┘                └──────────────────┘
│         │                                    │
│         └──── TCP 127.0.0.1:3306 ────────────┤
│              (MySQL 数据库连接)              │
│                                              ▼
│                                      ┌──────────────────┐
│                                      │    MySQL 5.7.8+  │
│                                      │   (数据持久化)    │
│                                      └──────────────────┘
│
│ ┌──────────────────────┐                ┌──────────────┐
│ │   freecdn-node       │────gRPC 8003──→│ freecdn-api  │
│ │  (边缘节点进程)      │  endpoint      │              │
│ │ 多进程分布式部署     │                └──────────────┘
│ └──────────────────────┘
│
│ ┌───────────────────────────────────────────────────────┐
│ │        EdgeCommon（共享库 - Go module）               │
│ │ 包含：pb/ dao/ protos/ nodeconfigs/ serverconfigs    │
│ │ module 名：github.com/hujiali30001/freecdn-common    │
│ └───────────────────────────────────────────────────────┘
└─────────────────────────────────────────────────────────┘
```

### 模块说明

| 模块 | 职责 | 技术栈 |
|------|------|--------|
| **freecdn-admin** | Web 管理后台（UI、路由、表单） | Go + Vue 2 + Semantic UI + LESS |
| **freecdn-api** | 核心 gRPC 服务（数据操作、认证、节点管理） | Go + gRPC + MySQL |
| **freecdn-node** | 边缘节点进程（流量转发、缓存） | Go + CGO（mattn/go-sqlite3） |
| **freecdn-common** | 共享库（protobuf 定义、DAO 接口、配置结构） | Go module |

### 已知坑清单
- 所有配置文件权限必须是 `0600`（包含 RPC 密钥）
- `edgeAPINodes.http.listen[0].protocol` 必须是 `"http"`（非空字符串）
- `api_admin.yaml` 必须用嵌套 YAML 格式，不能用点号格式
- `edgeAPINodes.accessAddrs` 必须包含真实 IP（不能为空）
- edge-api 和 edge-admin 均不直连 MySQL，所有操作通过 gRPC API
