- **圆角**: 20px (卡片) / 8px (按钮)

- **间距**: 24px (桌面) / 16px (平板) / 12px (手机)
- **悬停**: translateY(-4px) + 阴影增强
- **过渡**: all 0.3s ease

- **桌面**: >1400px (4 列 + 双列图表)
