📲 移动端（<768px）：1列卡片 + 全屏图表

## 🚀 快速开始

### 第 1 步：编译 CSS
```bash
cd web/views/@default
lessc @modern_theme.less @modern_theme.css
```

### 第 2 步：导入样式
在 `@layout.html` 添加：
```html
<link rel="stylesheet" href="/_/@default/@modern_theme.css">
```

### 第 3 步：测试页面
- 访问 Dashboard 页面验证效果
- 在浏览器开发工具中测试响应式
- 检查所有图表正常显示

### 第 4 步：应用到其他模块
- 复制相关 HTML 结构
