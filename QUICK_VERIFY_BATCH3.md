## 🔍 常见问题快速排查

### ❓ CSS 文件没有加载 (404)
```
✓ 解决: 检查文件是否在指定位置
  ls -la /mnt/c/Users/Administrator/.workbuddy/FreeCDN/web/views/@default/index/index.css
  
✓ 如果文件不存在: 重新运行编译
  node compile_less.js
```

### ❓ 页面加载但没有样式 (白色页面)
```
✓ 解决: 
  1. Ctrl + Shift + Delete 清除浏览器缓存
  2. F5 硬刷新 (Ctrl + Shift + R)
  3. 重启浏览器
```

### ❓ 样式不完整或错乱
```
✓ 解决:
  1. 检查浏览器控制台是否有 LESS 编译错误
  2. 运行: node compile_less.js 查看错误信息
  3. 检查 LESS 文件是否有语法错误
```
