## 🚀 即刻可用的快速开始

### 3 步快速部署
```bash
# 步骤 1: 编译主题
cd web/views/@default
lessc @modern_theme.less @modern_theme.css

# 步骤 2: 导入到主布局
# 编辑 @layout.html，在 <head> 中添加：
# <link rel="stylesheet" href="/_/@default/@modern_theme.css" media="all"/>

# 步骤 3: 启动验证
# 1. 启动服务
# 2. 访问 Dashboard
# 3. 按 F12 检查样式
# 4. 缩放窗口测试响应式
```
